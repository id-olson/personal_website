Lucky Lightning!

This was another really engaging project. I made multiple changes to how the lighting moved and appeared, which led to lots of troubleshooting with variable names and formatting. I also wanted to add an aspect of luck and engagement, so I wanted to add text to the screen, display it at a random position, track whether the lightningâ€™s path intersected with the text, and then have some sort of result when it does intersect that includes how many tries it took to get the text and lightning to line up. The hardest part of this process was figuring out how to carry out each step in code, and I solved it by thinking through what information the computer needs to know to do what I want it to do. There were also some issues with clearing the screen when I wanted and displaying text using the draw() function, but I was able to solve these in java using trial and error.
