Daring Dodgy Dots Defense Display! ~ Chemotaxis Project

This project provided a very useful application of multiple aspects of object-oriented programming with classes. I gained a better understanding of constructors, parameters, and accessors. I also became more familiar with the draw( ) method, and I used multiple debugging methods to test that my code worked correctly. One part of the project that I struggled with was converting the java code into p5.js because there were many private instance variables in the class that had to be moved in order to compile in the p5.js version. I ended up using the this. keyword to instantiate instance variables of the object type that I needed for this project. The this. keyword solved my p5.js conversion problems because p5.js is able to execute many more implicit functions than java.
