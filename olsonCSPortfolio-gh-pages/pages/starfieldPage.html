Circles in Motion ~ Starfield Project

This project helped me understand the benefits of interfaces and inheritance, and it reinforced my familiarity with objects, parameters, and methods. I found an online example of code that replicated circular motion, and that helped me understand how cos() and sin() can be used to model a circle. I realized that by changing an objectâ€™s position by the cos() and sin() of a given angle and then incrementing the angle, the object will move in a circle. Thinking through the cos() and sin() values for various angles reminded me of the unit circle, which I learned about in algebra class. For example, I realized that this code works because even as the angle becomes greater than 2 pi, the object will continue moving in a circle because the sin() and cos() at pi radians are equal to the sin() and cos() at 3 pi radians, for example. After I understood how this code worked, I was able to adapt the parameters it uses to create variation in the size and speed of the circles in my project.
