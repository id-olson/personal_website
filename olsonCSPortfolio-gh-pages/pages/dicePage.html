Very Cool Dice!

This was a very useful review of object-oriented programming. It was challenging to figure out how to declare objects, add parameters to the constructor, and call the right methods from the Die class to carry out each task. I was able to figure out the syntax and formatting by thinking through what data I needed to pass in at each step. For example, I knew that each Die object I created would have an x-value and y-value to help position it. Therefore, I declared private instance variables for these values in the Die class, included two int parameters in the Die constructor, and assigned each Die object I created values for each of the parameters. This approach worked because it allowed Die objects to pass in 2 values for positioning. Converting it to javascript was another challenge, but I learned a lot about classes, methods, and private instance variables in javascript in the process of figuring out the conversion.
