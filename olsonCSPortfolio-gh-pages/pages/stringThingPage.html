String Thing!

This project involved using various String properties in peblio. One challenging part of this project was converting the code from the java that peblio uses to p5.js, which is javascript. However, I used this website to automatically update my p5.js code, which made it much easier to figure out where exactly the problem was in my code. It turns out that I needed to re-write the code for how I change fonts. This process involves creating a PFont object in java, but it is much simpler to do in p5.js. After I changed that, I just had to alter how I labeled private instance variables and the names of methods, and my original java code was fully converted to p5.js.
